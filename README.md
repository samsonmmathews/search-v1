# Search V1

This application will provide a seach engine experience for BrickMMO and CodeAdam assets. 

> The live website is available at:  
> https://search.brickmmo.com

## Future Search Enhancements

The following additions can be made to search results and the spiders:

- Search Events
- Search Roads
- Search places
- Colour profile if search is exact colourColours
- Part/Swet/Theme if search is exact match
- Easter eggs
- Search GiHub, Instagram, TikTok, and YouTube
- Search Applications
- Search Brix
- Search Contributors (can be added to spiders)

---

## Project Stack

This application will intentionally be written in a simplified structure with a simple stack. This will provide BrickMMO developers with a simple application to contribute to once they have some understanding of server-side languages such as PHP.
